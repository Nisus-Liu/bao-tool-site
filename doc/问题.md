- [x] 数组内 {} 元素时, 改数组后解析给了数组内的对象了

```
{
  "name": [{}],
  "age": 123
}
```



- [x] 注释丢失

symbol 类的 value , 且跟上',' 就不行, 没有','可以.

```
{
 
  // 是否是坦克
  "isTanke": true,
}
```

```
{
  // 姓名
  "name": "张飞",
  // 年龄
  "age": 123,
  // 经济
  "money": 98700.123,
  // 是否是坦克
  "isTanke": true,
  // 生日
  "birthday": "2022-3-27 23:39:45"
}
```

- [x] '}' 后面的注释归到了最后个 item, 且不应该包含 EOI 字符(显示乱码).

最后 item boolean 能复现.
```
{
  // sag
  "abc" : true
}//tail
```
